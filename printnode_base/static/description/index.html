Odoo Direct Print (PrintNode)

VentorTech

Print any reports or shipping labels directly to any local, Wi-Fi or Bluetooth printer without downloading PDF or ZPL!

Description

The app connects Odoo to your PrintNode account and links all the printers you already configured there.

Click here to create a new PrintNode account for just 1 minute.

Each printer can be configured per user. All this allows you to print much faster compared to the default Odoo printing method. Now you can print packing slips (PDF) on your regular A4 printer and shipping label (ZPL) on any thermal Zebra printer all at the same time. You can also configure automatic print while calling some action (beta), e.g you can print SO right after click "validate" button.

Features

TEST ON OUR SERVER

Our Demo server is recreated every day at 12.00 AM (UTC). So all your manually entered data will be deleted.

  • Working with both Odoo Community and Enterprise (self-hosted or odoo.sh)
  • Automatic printing of different document types (ZPL, PDF) on different printers (configurable per user if needed), no need to download files
  • Smart validation not allowing users to print documents on unsupported printers (not allowing to print A4 docs on A6 printers, or PDF on ZPL printer and etc.)
  • Support of USB, Ethernet, WiFi, Bluetooth Printers
  • Flexible automatic printing configuration (e.g. print Sales Order PDF automatically after clicking on Confirm button) (BETA)
  • All kind of reports and paper size
