Backend Stripe Payment

In Odoo standard modules online payment options is only available on website for portal customer.This module allows Admin/Account user to pay using stripe from back end.

This module is compatible with.

  • Community
  • Enterprise
  • Odoo.sh

Access Rights

Go to user profile & Allow user to 'View Online Payment Options'.


Stripe Configuration

On Invoice Payment option set 'Automatic pay invoice on acquirer confirmation' & set related Payment Journal.


Customer Invoice Payment

Once customer invoice goes to 'Open' stage pay with stripe option is visible on invoice.


Stripe Checkout Box

Customer Info & Total invoice amount is reflect on stripe checkout box.


Invoice Payment

Once user will click on 'Pay' then new payment is created & invoice gose to 'paid' stage automatically.


Stripe Payment

Payment Transaction shows payment amount & reference.


Invoice Report

It is also possible to directly pay online by pay now button in invoice report.


Customer Invoice Payment

Once click on paynow button you will be redirect to website customer invoice payment.


Stripe Checkout Box

Customer Info and Total invoice amount is reflect on stripe checkout box.


Invoice Payment

Once user will click on 'Pay' then new payment is created and invoice gose to 'paid' stage automatically.
